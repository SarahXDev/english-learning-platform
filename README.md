# English Learning Platform 

An interactive web-based platform designed to help users improve their English skills, with a focus on speaking, listening, reading, and writing.

##  Project Type
This project is a **demo / prototype** developed for learning and demonstration purposes.

##  Features
- 🎤 Speaking practice with microphone support  
- 🎧 Listening exercises  
- 📖 Reading section  
- ✍️ Writing practice  
- 📊 Progress tracking  
- 🧠 Multiple topics for speaking training  

## Technologies Used
- HTML  
- CSS  
- JavaScript
